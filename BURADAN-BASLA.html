Bu siteye ulaşılamıyor / Bağlantı reddedildi?

Tarayıcıda yalnızca 127.0.0.1 veya localhost yazarsanız bilgisayar 80 numaralı porta bağlanır. Spor Okulu burada çalışmaz; o yüzden ERR_CONNECTION_REFUSED görürsünüz.

Doğru adresler (sunucular çalışırken):
http://127.0.0.1:3000/giris
API dokümantasyonu: http://127.0.0.1:8020/docs

Kolay yol:

  1. BASLA.bat ile aynı klasörde olun (proje kökü: içinde frontend ve backend var).
  2. BASLA.bat dosyasına çift tıklayın.
  3. İki siyah pencere açılır; kapatmayın. Tarayıcı, port 3000 hazır olunca açılır.
  4. Hâlâ “bağlantı reddedildi” ise BAGLANTI-KONTROL.bat çalıştırın; Web (3000): False ise Web penceresindeki hatayı okuyun (Node kurulu mu, npm install oldu mu).

Bu sayfa (BURADAN-BASLA.html) sunucu gerektirmez; sadece yönergeler içindir. Asıl uygulama yine BASLA.bat ile çalışmalıdır.
